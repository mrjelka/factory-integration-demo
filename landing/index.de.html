Protokolle & Technologien
OPC UA
Industriestandard-Protokoll. Server bildet Drucker, Scanner, Sensor und Produktionslinie als Informationsmodell ab.
MQTT
Leichtgewichtiger Event-Bus. Maschinendaten werden auf Topics publiziert, beliebige Subscriber (MES, SPS, Cloud) konsumieren.
REST / HTTP
Steuerungs-API zum Starten, Stoppen und Auslösen von Fehlerszenarien. Control Panel kommuniziert per REST.
InfluxDB Flux
Zeitreihen-Speicher. Alle OPC-UA-Variablen werden als Messpunkte geschrieben und in Grafana visualisiert.
Docker Compose
Sechs Services, ein Befehl. Gesamte Infrastruktur containerisiert und reproduzierbar.
Node.js + Python
OPC UA Server und Data Bridge in Node.js, Control Panel in Python/FastAPI. Praxisnahe Kombination.
